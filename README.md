# Video Player in Swift

[![Swift](https://upload.wikimedia.org/wikipedia/commons/thumb/9/9d/Swift_logo.svg/382px-Swift_logo.svg.png?20210606004230)](https://www.swift.org/documentation/)

[![Build Status](https://travis-ci.org/joemccann/dillinger.svg?branch=master)](https://github.com/ShivadityaKr/VideoPlayer--iOS-Swift/tree/main)

This Video player is made-up with swift UiKit and AVFoundation

- Swift UiKit
- AVFoundation

##Screenshots

## Features

- Play Video 
- Play and pause functionality
- Seek Functionality
- Jump to next Video
- Load Video from URL





## Usages

Clone from git URL or download the project and open in your xCode and simply run the project.

There is no any dependency added to the project.
